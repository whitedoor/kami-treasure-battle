# カードの生成ロジック

* レシートの画像から商品名、金額、購入店舗を抽出して、カード名、画像、攻撃力、じゃんけん（ぐー、ちょき、ぱー）を生成する
* レアリティは攻撃力に応じて変わる
    * 20：ブロンズ
    * 30：シルバー
    * 40：ゴールド
    * 50：レジェンド
* 攻撃力は20, 30, 40, 50のどれかで生成する
* デフォルトで全ユーザーに攻撃力10のすべてのジャンケンカードは配布されている
    * レアリティはノーマル
* カード名は魔法っぽい名前で生成する
* 画像はカード名の魔法を唱えている画像にする

# カードの表示ロジック

* カードの枠組みは事前にCSSやJSで作成し、生成された画像を中心に表示するだけにする
