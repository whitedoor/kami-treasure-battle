# MVPスコープ（CPU戦 + コレクション + 日次報酬）

このMVPは「遊べる最小」を優先し、**CPU戦はオフライン**で成立させつつ、**所持カード・日次報酬はサーバで管理**します。

## MVPでやること

- **認証/アカウント**
  - Firebase Authenticationでサインアップ/サインイン
  - 初期プロフィールとして「じゃんけんキャラ（グー/チョキ/パー）」を保存
  - キャラは後から変更可能
- **カード所持**
  - カード一覧（所持/未所持、カード詳細）
  - バトルに使うカードを、事前に「グー/チョキ/パー各1枚」選択して保存
  - 3枚の合計攻撃力は100以下（保存時に検証）
- **日次報酬**
  - 1日1回（JST 0:00 リセット）
  - 「揃うまで重複なし」
  - 付与はサーバ側で原子処理（多重受取防止）
- **バトル（CPU戦）**
  - 毎ターン、3枚から1枚を選ぶ
  - 勝利: 相手HPをカード攻撃力分減らす
  - ユーザーが選択したじゃんけんで勝利: ダメージ1.5倍（四捨五入）
  - 相手HPが0で勝利

## MVPでやらないこと（将来）

- オンライン対戦（リアルタイム/非同期）
- マッチメイク、ランク、リプレイ、チート耐性強化
- 課金/ガチャ/ショップ
- カードの複数枚所持・強化・合成（必要なら後で追加）

## 暫定前提（ユーザー回答が未確定だったため）

- **あいこ時**: 両者が「初期体力の5%」を受け、次ターンへ（初期HP=200なので10ダメージ）
- **日次報酬の母集団**: `cards(active=true)` を対象に、未所持を優先して1枚付与
- **コンプリート後**: いったん「付与なし」または「別報酬」にする（要決定）
